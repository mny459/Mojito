# 一个Android快速开发框架

## 合理的分层

- Mojito：Mojito 框架提供的Base
- Base：与业务无关的通用、基础类，
- Common：与业务有关的通用封装，比如User等的操作
- Lang：多语言文字存放
- Res：统一资源文件、比如图片存放
- feature 下面是各个业务模块的具体内容
- buildSrc：统一第三方框架管理
